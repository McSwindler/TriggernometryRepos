# TriggernometryRepos
Repositories for [Triggernometry](https://github.com/paissaheavyindustries/Triggernometry)

## How to Add Repository
1. Right-Click Remote triggers
1. Add -> Repository
1. Enter Repository Name
1. Copy and Paste Address below
